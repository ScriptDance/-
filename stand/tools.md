# 下载与安装
 - [点击下载A](<https://scriptdance.github.io/Node-Script/assert/jdscript.apk>)
 - 暂且仅支持 Android 设备
     - ·真机 · 模拟器（雷电，夜神，mumu 等） 
 - 模拟器安装拖动进模拟器即可。

# 使用 节点精灵 查看设备节点
 - 真机
   - 请保持设备与PC电脑 同一个Wifi 网络
   - 在浏览器中输入 节点精灵主页 显示的ip 地址 访问即可
   - 在显示出的网页中点击 查看view树
 - 模拟器
   - 配置网络桥接，保持和PC电脑统一网段
   - 在浏览器中输入 节点精灵主页 显示的ip 地址 访问即可
   - 在显示出的网页中点击 查看view树
