# <a name="#1">创建检索规则</a>
- 建一个检索命令，检索命令支持链式编程增加规则内容
    ```
    R();
    ```

# <a name="#2">文本-检索</a>
- 创建一个规则，并给这个规则 增加文本检索。
- 检索界面上所有 文本未 ‘退出’的控件。
- 支持正则
    ```
    R():text("退出");
    ```

# <a name="#3">ID-检索</a>
- 创建一个规则，并给这个规则，增加id 检索
- 检索界面上 id 为‘com.touchscrip/id123’的控件。
- 控件id 可以用用浏览器查看 “设备ip:8080/dump”
- 支持正则
```
R():id("com.touchscrip/id123");
```
